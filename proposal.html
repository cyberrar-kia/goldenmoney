Design Approach

Minimalist. Custom.
Nothing borrowed.

Every design decision was made deliberately. Here's the thinking behind the visual approach:

Typography

Georgia Serif + System Sans

Headings use Georgia — the web's most trusted editorial serif. Body uses system-ui. No Google Fonts dependency needed for OpenCart.

Layout

Grid Lines, Not Cards

Sections separated by 1px lines and white space — not rounded cards or drop shadows. Minimalist, like a financial publication.

Structure

Long Scroll, Single Page

One continuous page. Ticker → Nav → Hero → Prices → Buy → Why → About → Help → Contact → Footer. Logical flow top to bottom.

Code

OpenCart-Compatible CSS

Standard CSS variables, no preprocessor, no utility framework. Drop into OpenCart's theme file without modification.

Interaction

Subtle, Not Showy

Hover states on cards, smooth scroll, sticky nav on scroll. No GSAP, no particles. The metal sells itself.
